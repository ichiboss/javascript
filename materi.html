Dasar JavaScript

1 Variabel & Tipe Data

Variabel adalah wadah untuk menyimpan data. JavaScript memiliki tiga cara mendeklarasikan variabel: var, let, dan const. Setiap variabel memiliki tipe data yang menentukan jenis nilai yang disimpan.

4 Fungsi

Fungsi memungkinkan kita mengelompokkan kode menjadi unit yang dapat digunakan kembali. JavaScript mendukung deklarasi fungsi, ekspresi fungsi, dan arrow function (ES6+).

Deklarasi Fungsi
function sapa(nama) {
  return "Halo, " + nama;
}
Ekspresi Fungsi
const sapa = function(nama) {
  return "Halo, " + nama;
}
Arrow Function
const sapa = (nama) => {
  return "Halo, " + nama;
}
// Singkat:
const sapa2 = nama => "Halo, "+nama;

Manipulasi DOM

Document Object Model (DOM) adalah antarmuka pemrograman untuk HTML. Dengan JavaScript, kita bisa memilih, mengubah, membuat, dan menghapus elemen HTML secara dinamis.

6 Memilih Elemen DOM

Sebelum memanipulasi elemen, kita harus memilihnya terlebih dahulu. JavaScript menyediakan beberapa metode selection yang berbeda.

Metode Hasil Contoh

getElementsByTagName()HTMLCollectiondocument.getElementsByTagName("p")

7 Mengubah Konten Elemen

Setelah elemen dipilih, kita bisa mengubah isinya menggunakan properti textContent, innerHTML, dan value (untuk input).

textContent
Mengubah teks murni. Aman dari XSS. Tag HTML tidak di-render.
innerHTML
Mengubah HTML termasuk tag. Tag akan di-render. Hati-hati XSS!
value
Membaca/mengubah nilai input form (text, select, dll).

Event Handling

Event adalah aksi yang terjadi di halaman web — klik, ketik, hover, dan lainnya. Dengan event handler, kita membuat halaman merespons interaksi pengguna.

10 Event Dasar

Ada tiga cara mendaftarkan event handler: inline attribute, properti DOM, dan addEventListener() (yang paling direkomendasikan).

CaraSintaksKelebihan / Kekurangan
inline <button onclick="fn()"> Simpel tapi mencampur HTML & JS
DOM property btn.onclick = fn Hanya 1 handler per event
addEventListener btn.addEventListener("click", fn) Bisa banyak handler, bisa di-remove
